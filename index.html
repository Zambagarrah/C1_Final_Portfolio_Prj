About Me

My Journey

HTML

My journey began with HTML, the backbone of any webpage. I see it as the structure, the skeleton. It defines the content: the headings, paragraphs, images, and links. I learned how tags create this structure, allowing the browser to understand and display information in an organized way.

CSS

Next came CSS, the style master. CSS taught me how to make the website to be visually appealing. I understand how to use CSS to control the appearance of the HTML elements: the colors, fonts, layout, and overall visual design. It's like giving the skeleton a stylish outfit!.

JavaScript

Finally, i met JavaScript, the interactive powerhouse. This is where things get dynamic! JavaScript allows me to understand how to make webpages interactive and responsive. I learned how to make elements move, respond to user actions, and update content on the fly. It's like giving the webpage life!

In summary, HTML provides the content, CSS styles it, and JavaScript adds the interactivity. Together, they create the amazing web experiences we see today!

Callanges faced throughout my journey

  1. HTML
    • Semantic Understanding: Grasping the nuances of semantic HTML (using tags like: nav & article) to improve accessibility.
    • Browser Compatibility: Dealing with different browsers rendering HTML slightly differently, requiring careful testing and adjustments.

  2. CSS
    • Specificity: Understanding the CSS specificity rules and how to override styles effectively.
    • Layout Challanges Mastering complex layouts using techniques like Flexbox and Grid, which can be tricky at first.
    • Cross-Browser Issues: Addressing incosistency in how different browsers interpret CSS properties.

  3. JavaScript
    • Debugging: Learning to debug JavaScript code effectively, which can be a time-consuming process.
    • Asynchronous Programming: Understanding asynchronous operations (like fetching data from a server) and how to handle them correctly using Promises or async/await.
    • Frameworks and Libraries: Keeping up with the ever-evolving landscape of JavaScript frameworks (like React, Angular, or Vue.js) and libraries.
